ese_cardboard
=============

googleのcardboardもどきです。

100円ショップのルーペと、両面テープと、amazonの宅配に使われる箱の底に敷かれているダンボール2枚でつくることができます。


ルーペはloupe.jpgにあるようなものを使います。分解するとレンズが2枚入っていますが、片目につき1枚ずつ使いますので1つ買えばいいです。

焦点距離は約55mmで、これにあわせて設計しています。

ダンボールの大きさは297mm x 240mm、3mm厚を想定しています。


組み立て
========

ese_cardboard_sheet1_line.svg, ese_cardboard_sheet2_line.svgの通りにダンボールをカットします。

assembly1.jpgのようにレンズをダンボールではさみます。

assembly2.jpgのように組み立てます。

assembly3.jpgのように両面テープを貼り付けます。

assembly4.jpgのように組み立てて完成です。


使い方
======

スマホを後ろに入れ、ふたを閉めます。
スマホは15mmの厚さまで対応するように作っていますので、イマドキの薄いスマホを使う場合は余ったダンボールで隙間を埋めるなど工夫してみてください。

鼻先にあるセパレータが焦点の合う距離になっています。左右で同じ距離になるように後ろの穴から指を使って調整します。左右の位置も同じように調整できます。
